Projeto em Java

Ler um arquivo texto e marcar cada linha com OK ou Inválido:

{[]} - OK

([)] - Inválido

[{()()}[]] - OK

{}()[()]] - Inválido

)[{}]()( - Inválido

(()[)] - Inválido

etc...

Criar um programa java que recebe um arquivo texto com código via linha de comando, exemplo:

java gera prog.txt 
saída -> prog-check.txt
